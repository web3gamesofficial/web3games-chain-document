## Description

CLI tool to work with Web3Games chain

## Installation

```sh
npm install -g @web3games-js/cli
```

or

```sh
yarn global add @web3games-js/cli
```

## Usage

#### Full list of commmands

```sh
web3games --help
```

#### Available commands

- **login** - _web3games wallet login in_


#### Examples

```sh
web3games login 

# action:
  # openUrl https://w3wallet.vercel.app/
```
